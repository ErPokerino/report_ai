# Cartella Contesto

Questa cartella contiene file di contesto che forniscono informazioni sul progetto, sui dati e sulla conoscenza di dominio specifica.

## Scopo

I file in questa cartella servono a:

- **Fornire contesto al sistema AI**: I file possono essere referenziati nei prompt AI per migliorare la qualità delle analisi, utilizzare terminologia corretta e fornire conoscenza di dominio specifica.

- **Documentare il progetto**: Informazioni sul sistema, sui dati e sul dominio di applicazione.

- **Supportare decisioni analitiche**: Conoscenza di dominio che aiuta a decidere quali analisi effettuare e come interpretare i risultati.

## Tipi di File

La cartella può contenere vari tipi di file:

- **File Markdown (.md)**: Documentazione testuale, dizionari dati, conoscenza di dominio
- **File PDF**: Documentazione formale, specifiche tecniche, diagrammi di flusso
- **File di testo (.txt)**: Note, annotazioni, informazioni aggiuntive
- **Altri formati**: Qualsiasi file che fornisce contesto utile al progetto

## Contenuti Tipici

I file possono includere:

- Descrizione del sistema e del dominio applicativo
- Dizionari dati e metadati
- Documentazione dei flussi di processo (es. flusso invoice, machine learning pipeline)
- Terminologia specifica del dominio
- Informazioni sul progetto e sul contesto aziendale
- Specifiche tecniche e architetturali

## Utilizzo

Questi file possono essere:

1. **Letti manualmente** per comprendere il contesto del progetto
2. **Referenziati nei prompt AI** per migliorare le analisi automatiche
3. **Aggiornati, aggiunti o rimossi** man mano che il progetto evolve

## Note Importanti

- **I file sono dinamici**: Possono essere aggiunti, modificati o rimossi in base alle esigenze del progetto
- **Formati multipli supportati**: La cartella può contenere file in vari formati (MD, PDF, TXT, etc.)
- **Informazioni sensibili**: Verificare sempre il contenuto prima di committare, alcuni file potrebbero contenere informazioni riservate
- **Manutenzione**: Mantenere i file aggiornati con le informazioni più recenti sul sistema
